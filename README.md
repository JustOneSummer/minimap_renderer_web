# 安装前准备

安装git

安装python环境 python版本3.10

`LLOneBot` 插件版本必须大于等于 `v3.31.6`

`go-cq-http` 用户需要配置 `ffmpeg` 

文档： https://docs.go-cqhttp.org/guide/quick_start.html#%E5%AE%89%E8%A3%85-ffmpeg

如果非同源本地部署 LL bot请打开这个设置
